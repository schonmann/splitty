Splitty is a simple static file server with a embedded text editor based on ACE. splitty is focused to speed up the web development allowing the programmer to use de ACE editor on the left side and see the result on the right side of the same browser tab.

Download as .zip Download as .tar.gz View on Github

Text editor, Shell and Web visualizer

In a single browser tab you can edit files in a editor based on ACE, visualize your app running on browser and execute shell commands

Dependencies

NodeJS 4+
NPM

Install Splitty

$ npm install [-g] splitty

Run

$ cd PATH_AS_YOU_WANT_TO_SERVE
$ splitty [port=8000]

Open Splitty Editor

http://localhost:8000/_editor
